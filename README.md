<h1 align="center">Discussions</h1>

This repository is the home of GitHub Discussions for **The Lupaxa Project**.

It provides a central place for:

- Asking questions
- Proposing ideas and improvements
- Sharing designs, concepts and early explorations
- Discussing project direction and ecosystem standards
- Community discussions

All discussion threads, categories and related metadata are stored in this repository. No source code is maintained here.

## The Central Discussion Hub

Use these discussions for topics that span multiple organisations, or relate to **The Lupaxa Project** as a whole.

Organisation-specific discussions should normally take place within the discussions repository of the relevant organisation.
